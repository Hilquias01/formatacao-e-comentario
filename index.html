Guia de Tags de Formatação de Texto

        A tag <b> é para negrito (apenas visual).
        A tag <strong> é para texto com grande importância (semântico). 
        A tag <i> é para itálico (usado para termos técnicos, pensamentos, etc.). 
        A tag <em> é para dar ênfase ao texto (semântico). 
        A tag <mark> serve para marcar ou realçar um texto. 
        A tag <small> é para textos com menor importância (como direitos autorais). 
        A tag <del> mostra um texto como deletado ou removido. 
        A tag <ins> mostra um texto como inserido ou adicionado. 
        A tag <sub> é para texto subscrito (ex: a fórmula da água é H₂O). 
        A tag <sup> é para texto sobrescrito (ex: para elevar um número ao quadrado, como em X²).
